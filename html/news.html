Hello Universe! [4.28.09]

Welcome to Spaceship! This game is meant to be a fun activity for kids of all abilities, a way for them to learn some basic computer skills, and a way for us to push Web browsers to new heights. Please, give it a try.

This first version lets gamers use the mouse or keyboard and includes graphics, speech, and sound. We're working on improvements to support joysticks, more gameplay options, a wider variety of mini-games, and whatever cool suggestions we get from you. Don't hesitate to contact us!

Choose the Help option on the main menu to learn how to play. Or, if you're brave, just pick New Game and dive right in.

Happy gaming!
